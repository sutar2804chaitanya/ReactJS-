# ReactJS-# ⚛️ ReactJS Hooks and Context API Demo

A ReactJS project demonstrating the practical implementation of React Hooks and Context API. The project showcases state management, event handling, conditional rendering, image manipulation, dynamic styling, and data sharing between components using Context API.

## 🎯 Aim

To implement ReactJS Hooks and Context API concepts for building interactive and reusable user interface components.

## ✨ Project 1: React Hooks Demonstration

This application demonstrates the use of React Hooks for managing component state and user interactions.

### Features

* Dynamic text change using `useState`
* Image switching functionality
* Dynamic color changing
* Conditional rendering of content
* Event handling with buttons
* Show/Hide component functionality

### React Concepts Used

* Functional Components
* `useState()` Hook
* Event Handling
* Conditional Rendering
* Dynamic Styling
* State Management

## ✨ Project 2: Context API Demonstration

This application demonstrates how data can be shared across components without prop drilling using React Context API.

### Features

* Global data sharing using Context API
* Data access using `useContext()`
* Nested component communication
* Clean component structure

### React Concepts Used

* `createContext()`
* `Context.Provider`
* `useContext()`
* Component Hierarchy
* State Sharing
